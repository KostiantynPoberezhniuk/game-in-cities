
# Гра в міста
Класична гра "Міста" українською мовою з графічним інтерфейсом (Swing).
Гравець змагається проти комп'ютера — по черзі називають українські міста,
кожне наступне місто має починатися на останню літеру попереднього.
## Вимоги
- Java 21+
- Gradle 9.3.1 (або використовуйте вбудований Gradle Wrapper)
## Запуск
через клас Main.java
src/main/java/Main
=============================================================================
## Правила гри:
1. Гравець першим називає будь-яке українське місто.
2. Комп'ютер відповідає містом, яке починається на останню літеру попереднього міста.
3. Букви "ь" та "й" у кінці міста пропускаються — береться попередня літера.
4. Кожне місто можна використати лише один раз.
5. Гравець може написати "здаюсь", щоб завершити гру.
6. Якщо комп'ютер не знаходить місто — гравець перемагає.
7. Після завершення гри відображається підсумковий рахунок.

Як змінити список міст
Список міст зберігається у файлі:
src/main/resources/cities.txt

Кожне місто — на окремому рядку. Щоб додати або видалити місто, просто відредагуйте цей файл.
Порожні рядки та пробіли на початку або в кінці рядка ігноруються.
================================================================================

└── cities.txt                     — список міст
